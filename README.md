# Huestun-Introduction-Problems
This is a repo I made in order to on-board interns and new engineers for our company with example problems that are designed to introduce someone to some of the neccesary projects and conecpts needed. Some of these problems are oriented towards someone with little/no programming experience, and some are oriented towards a more experienced dev. All of these problems were designed and written completely without the use of AI, and AI use is generally discouraged in solving these (unless noted in some specific problems) as it will detract from the learning experience when you are building an understanding of some of these core concepts. As you progress through them, it will become more relevant.
